第一次成功用arduino点灯

起因

昨天心血来潮网购了个Arduino套件，没想到今天就到了。迫不及待试了一下，完成了点灯仪式 🥳

过程

先是安装 Ardunio IDE，这里就不赘述了。设备识别也很顺利，

由于我对面包板还不是很熟（到现在上的实验课还没接触到），接线的过程稍微费了点时间，但是理解了面包板内部哪些地方是导通了之后实际上接线也没那么复杂。使用杜邦线把220Ω电阻与LED串联，短脚接GND，另一端接D13，整个电路就基本上完成了。

点灯程序也不复杂，直接看代码也能看懂，况且现在ai写代码的能力已经很强了，vibe coding 一些简单的程序都能在一瞬间完成。

// setup() 程序启动时执行一次
void setup() {
  pinMode(13, OUTPUT);  // D13 设为输出
}

// loop() 循环执行
void loop() {
  digitalWrite(13, HIGH);  // LED 亮
  delay(1000);             // 等 1 秒
  digitalWrite(13, LOW);   // LED 灭
  delay(1000);             // 等 1 秒
}

写好代码上传程序就可以了，看到LED闪烁的那一刻还挺开心的，感觉像打开了一个新的世界。

结语

作为一个电子学生，从大一开学到现在一直在vibe coding，还是有一种错位感的。这次点灯或许作为一个新的起点。

但是我微积分还没开始复习（预习），再不突击就要完蛋了…

等期中过后再进一步探索吧。
